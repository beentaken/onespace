Multiple Locations
When enabled, the add-on supports multiple group chats for each Web page at the same time (for example, by opening multiple browser windows or opening multiple tabs). Without that option, you visit only one group chat at a time.
Auto-Tracking
When enabled, the chat view always shows the group chat of the currently view Web page. Thus, when you select a different tab in the same or another browser window, the chat view automatically switches to curresponding group chat. Without that option, you have to select the group chats mannually.
